Python subprocess

Python的subprocess模块可以创建管道进行交互，适用于大部分黑框程序，但是速度较慢。

Python subprocess + threading

引入了多线程机制，速度快了不少。

Frida RPC

直接调用内部函数，挺快的

Appcall

Appcall是IDA5.6新增的一个功能，使用Appcall调用一个函数相当于执行以下步骤

  1. 保存寄存器现场
  2. 序列化参数
  3. 修改寄存器直到满足调用要求
  4. 修改ip为需要调用的函数地址
  5. 修改返回地址至特定的一段内存空间，该处存在断点，完成函数调用后暂停程序。
  6. 运行程序
  7. 反序列化参数，保存返回值
